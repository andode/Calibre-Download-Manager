# Calibre Download Manager
Vibe coded slop I (watched be) made to fix an issue I had getting files from Calibre Content Server into vaious folders for various media types on Android. 

The app lets you select a folder for audiobooks, comics, and books independently. It will then connect to a non-authenticated Calibre Content Server using OPDS and let you download into the corresponding folder. Typical ebook files go in Books, comics in Comics, audio files in Audiobooks. There's probably some overlap like PDF which goes to comics. I made this for me, with the implications that has, like not needing authentication since I connect via Tailscale anyways.
